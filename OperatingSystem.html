OPerating System

Software program that interacts with the computer and user.
Controls computer hardware, Manages system resources, supervises interaction between computer and user.
Operating system also forms a baae on which applications software is developed and executed.

Functions of an Operating System

  • Command Interpretation : CPU cannot understand the commands keyedin by user,
    it is the function of the Operating system to to translate these commands into language that CPU can understand

  • Perpheral Management :Operating system has to take care of the device(peripheral)
    attached to the system.Communication between these devices and the CPU is over seen by Operating System.

  • Memory Management: The OS handles the extremely important job of allocating memory
    for the various process running on the system.It also has to clean up after a process-i.e dispose of left-over data in memory.

  • Process Management: This is required if several programs hve to run at the same time.
    CPU time would then have to be rationed out by the OS to ensure that no program gets more than its fair share of CPU time.
